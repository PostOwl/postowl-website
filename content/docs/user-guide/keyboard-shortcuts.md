Keyboard shortcuts only work for signed in users.

### General shortcuts

- `h` go to home page
- `n` new letter
- `m` toggle menu modal
- `esc` close menu modal
- `f` go to friends page

### Editing shortcuts

- `cmd / ctrl + e` - edit letter
- `cmd / ctrl + s` - save / send letter
- `cmd / ctrl + enter` - save / send letter
